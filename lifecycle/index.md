# 生命周期

DevOps 生命周期覆盖从规划到监控的全流程。

- [规划](plan.md) — 需求与任务管理
- [开发](code.md) — 使用 `code` 命令管理子模块
- [构建](build.md) — 使用 `build status` 查看构建状态
- [测试](test.md) — 使用 `test status` 查看测试状态与覆盖率
- [发布](release.md) — 使用 `release publish` / `release status` 管理版本发布
- [部署](deploy.md) — 持续部署
- [运维](operate.md) — 日常运维
- [监控](monitor.md) — 运行监控
